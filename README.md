# Аналитический дашборд для организации конференций TED

Данный проект представляет собой интерактивный дашборд, разработанный в [Tableau/Power BI/другая BI-система], для анализа исторических данных о выступлениях на TED. Цель — помочь организаторам проводить запоминающиеся и интересные мероприятия на основе данных.

## Ключевые бизнес-вопросы

Дашборд отвечает на критически важные для организаторов вопросы:
*   **Популярные темы:** Какие темы наиболее востребованы и как на них реагирует аудитория?
*   **Структура события:** Каково среднее количество выступающих на конференции и средняя длительность выступления?
*   **Выступающие:** Кто из спикеров и представителей каких профессий вызывает наибольший отклик у зрителей?

## Функциональность дашборда

*   **Обзор данных:** Общие метрики (количество конференций, выступлений, уникальных спикеров и тегов).
*   **Анализ конференций:** Детализация по странам с показателями средней длительности, максимальных просмотров, общего количества аплодисментов и числа выступающих.
*   **Рейтинги выступлений:** Топ-10 самых смешных выступлений и топ-20 самых популярных тегов.
*   **Гибкость анализа:** Возможность детализации до информации по каждому выступлению.
*   **Интерактивные фильтры:** Селекторы по стране, названию конференции, тегу и дате записи выступления.

**Задача:** Предоставить инструмент для принятия данных-ориентированных решений при планировании конференций в духе TED.
